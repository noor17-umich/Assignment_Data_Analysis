# Data Analysis Assignment

This repository contains the R script for a data analysis assignment.

## Tasks

### Task 1: Data Aggregation
- Aggregated sales data by state using `aggregate()` and `dplyr`.

### Task 2: World Cup Matches Data Analysis
- Analyzed the World Cup data, including calculating unique locations, average attendance, and total goals scored by home teams.

### Task 3: Metabolites Data Analysis
- Processed metabolites data to handle missing values and analyze Alzheimer's patients.
